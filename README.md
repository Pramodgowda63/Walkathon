# 🚶 Walkathon

A modern **Walkathon Management System** built with **Python Flask, SQLite, HTML, CSS, and JavaScript**. It enables organizations to manage wellness events with user registration, team management, daily step tracking, live leaderboards, and an admin dashboard.

---

## ✨ Features

- 🔐 User Authentication
- 👥 Team Management
- 🚶 Daily Step Tracking
- 🏆 Live Leaderboard
- 📊 Admin Dashboard
- 📷 Photo Upload & Gallery
- ⚙️ Event Management
- 📱 Responsive Design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Pramodgowda63/Walkathon.git
cd Walkathon
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open in your browser:

```
http://127.0.0.1:5000
```
