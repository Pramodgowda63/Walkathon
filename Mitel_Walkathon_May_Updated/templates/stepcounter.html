Important browser limit

This web version can auto-sync while the page stays active, but mobile browsers do not keep motion sensors running after the browser is fully closed or the phone suspends the page. For true all-day background tracking you still need Fitbit, Apple Health, Google Fit, or a native app build.

How this version works

1. Start walking and allow motion plus location if the phone asks.

2. The app waits for a real walking rhythm before counting, so random shakes are less likely to become steps.

3. Auto save updates the server during the session. Manual save keeps the count on the page until you confirm it.

Best results

Keep the phone in a pocket or steady hand while walking.

Leave GPS assist on when possible. Indoors, GPS may be weak, so the tracker falls back to motion rhythm.

Tap the status pill to open debug details if you want to inspect sensor behavior.
